# Basic Router

## Step 1 Create Project
```bash
npm create vite .
npm install
npm run dev
```

## Step 2 Create Push to Github
```bash
git init
git add .
git commit -m "init"
git push
```

### when update code

```bash
git add .

```

## Step 3 Install React Router
```bash
npm i react-router
```

## Step 4 Global state with Zustand
```bash
npm install zustand
```

## Step 5 Fetch data
```bash
npm install axios
```

## Step 6 UI Dayisyui and Icon Lucide
```bash 
npm i -D daisyui@latest
npm install lucide-react
```
